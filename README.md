# 关于江哥

![江哥的个人博客](https://blog.xspeedup.com/logo.png "江哥的个人博客")

你好，很高兴你能来到我的个人博客。

我是**江哥**，一名完全依靠自学入门网页技术的普通爱好者，并非计算机科班出身。
从零基础摸索语法、反复实操调试，一路走来所有关于网页开发的知识，都是靠自己一点点沉淀积累。搭建这座博客小屋，初衷只是为自己打造一处永久的线上记录仓库。

## 📂 博客内容分为两大板块

### <span style="color:#2563eb">一、技术杂记｜代码成长</span>

收纳自学途中踩过的各类问题、网页实操笔记、代码复盘与学习心得。
一来方便日后遗忘时回头翻看梳理，二来也希望能给同样**零基础自学网页**的同路人，提供一点微薄的参考。

### <span style="color:#059669">二、生活随笔｜人间烟火</span>

记录日常见闻、突发感悟与平淡细碎的人间小事，当作存放生活碎片的自留地。

- - -

这里没有精心规划的栏目分区，也没有刻意打磨的爆款内容，文字随性、内容杂乱，**代码逻辑与烟火日常在此共存**。

不追逐流量，不迎合热度，仅仅想用文字与代码，完整留存自己一路走来的成长痕迹，慢慢记录，稳步前行。

如果你也是自学网页的同行，欢迎在评论区留言交流技术心得；
若是偶然路过、翻看生活随笔的访客，也愿你平安顺遂，万事舒心。
